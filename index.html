Política de Privacidade para GLA Companion

Última atualização: 28 de julho de 2025

Esta Política de Privacidade descreve como o aplicativo GLA Companion coleta, usa e compartilha informações.

1. Coleta e Uso de Dados

  • Compras no Aplicativo: Utilizamos o serviço da Google Play Billing para remover anúncios. Não armazenamos dados de pagamento.
  • Anúncios: Usamos Google AdMob, que pode coletar dados anônimos como o ID de publicidade.
  • Reconhecimento de Voz: Usamos os serviços do Android. Os dados são processados pelo Google.
  • Dados Locais: Preferências são armazenadas no dispositivo via SharedPreferences.

2. Compartilhamento de Dados

Compartilhamos dados com os seguintes serviços do Google: Play Billing, AdMob e reconhecimento de voz.

3. Segurança

Utilizamos serviços seguros do Google. Preferências são salvas apenas localmente.

4. Privacidade Infantil

Não coletamos dados de crianças menores de 13 anos.

5. Alterações

Atualizações serão publicadas nesta página.
